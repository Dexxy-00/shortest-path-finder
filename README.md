# Shortest Path Finder

A console-based C++ program that calculates the shortest path between cities.

## Concept

The program operates on a fictional world called **Aetheria**.

The dataset file `AetheriaMap.txt` stores distances between cities.  
Each entry defines a connection between two cities and the distance between them.

## Features

- Graph representation of cities and connections
- Shortest path calculation between any two cities
- Random map generation for the world of Aetheria
- Ability to manually add new cities or connections
- Persistent map data using `AetheriaMap.txt`

## Customizing the Map

You can modify the world map in two ways:

 **Manually edit the file**

   Edit `AetheriaMap.txt` to add or change connections.

## Technologies

- C++
- Graph algorithms
- File I/O
